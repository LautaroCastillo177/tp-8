6- ¿Qué clase de Bootstrap se utiliza para agregar un borde a un elemento? ¿Cuáles son sus opciones?

La clase que se utiliza para agregar un borde a un elemento es "border-". Las opciones disponibles son: border/border-top/border-bottom/border.right/border-left/border-start/border-end

Ejemplo: aywuieyui


7- ¿Cuál es la clase de Bootstrap que se utiliza para agregar un espacio interno al contenido de un elemento? ¿cuál era la su propiedad equivalente en CSS?

La clase que se utiliza para agregar un espacio interno al contenido de un elemento es "p-". Su propiedad equivalente en css es "padding"

Ejemplo: yaudyuias


8- ¿Qué clase de Bootstrap se utiliza para agregar un espacio externo alrededor de un elemento? ¿cuál era la su propiedad equivalente en CSS?

La clase que se utiliza para agregar un espacio externo alrededor de un elemento es "m-". Su propiedad equivalente en css es "margin"

Ejemplo: jasdhjsha


9- ¿Cuál es la clase de Bootstrap que se utiliza para aplicar sombras a un elemento? ¿cuáles son sus opciones?

La clase que se utiliza para aplicar sombras a un elemento es "shadow-". Las opciones disponibles son: "none (sin sombra)/sm (sombra pequeña)/lg (sombra grande)

Ejemplo: shadjhasjdaasdda
